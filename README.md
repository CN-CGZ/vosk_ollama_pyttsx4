# vosk_ollama_pyttsx4
中文简介 本项目是基于Python的智能语音助手，集成Vosk语音识别、大语言模型对话及语音合成。支持唤醒词激活、离线对话与多场景扩展，提供高可定制跨平台解决方案。  English Summary A Python-based voice assistant integrating Vosk speech recognition, LLM dialogue, and TTS. Features wake-word activation, offline interaction, and cross-platform customizability for diverse applications.
